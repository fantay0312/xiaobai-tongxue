# 小白同学 · 费曼反转式学习智能体

> 不是 AI 教你，是**你教会一个 AI**。

学生扮演老师，AI 扮演「困惑但会追问的学生」小白。通过梯度追问、误区注入与讲解质量评估，
把你讲不清的地方精确暴露成盲区，驱动「备课 → 讲解 → 赴考 → 批注 → 补学 → 再讲」的完整学习闭环——
教到小白出师那天，就是你真正学会之时。

**在线体验**：<https://tradingvane.com/xiaobai/>
（语音输入需此 HTTPS 入口；学习记录按账号存服务器，换设备登录即还原）

## 它是怎么工作的

- **导演状态机（纯代码）**：追问梯度 L1 澄清 → L2 举例 → L3 边界 → L5 迁移，途中按剧本注入误区试探；
  评估、状态推进永远是本地规则，LLM 只负责「理解讲解」与「生成台词」，任何一跳失败静默降级模板引擎，课堂不中断
- **评估证据化**：LLM 判定必须附「老师原话引文」且经归一化子串校验，杜绝幻觉命中；金句同样引文落地
- **泄漏出口守门**：小白的每句台词过检测——未解锁的术语禁止说出口，AI 学生不会"剧透"标准答案
- **防作弊层**：入口拦截套答案/角色反转/窃提示词；服务器网关锁死请求形状、按角色裁决温度与输出上限
- **故事系统**：拜师帖、旅程带、印章成就、心魔战报、小白日记、出师证书——学习履历长成一册可回看的成长册
- **语音输入**：课堂支持口述转文字（默认服务器代管密钥，可自配任何 OpenAI 兼容 `/audio/transcriptions` 端点）
- **按账号云存档**：登录即拉档、变化去抖推送、乐观并发防双端互踩；换台电脑记录不丢

内置 38 个可玩知识点：《大模型训练》6 讲 + 《操作系统原理》30 讲（蓝本南大 jyy 2026 春）+ 《Python 程序设计》2 讲，
每讲全密度配置：检查清单、误区剧本、摸底题库、备课材料包、延伸书单与示意图。

## 快速开始

```bash
cd app
npm install
npm run dev          # 前端(演示模式零依赖可跑,断网也能完整体验)
npm run simulate     # 教学引擎全量仿真回归(38 讲断言)
```

接真实 LLM / 起本地网关 / 生产部署 → 见 [`app/README.md`](app/README.md) 与 [`server/README.md`](server/README.md)。

## 仓库结构

```
app/                 # 前端:Vite + React 19 + TS,zustand,three.js(小白 3D 形象)
  src/engine/        # 教学引擎:评估/导演/渲染/守门/成就/回忆(纯函数,可 Node 仿真)
  src/data/topics/   # 38 讲课程数据(checklist/误区/题库/备课包/台词)
  src/pages/         # 门厅/备课/讲解舱/送考/复盘/成长册/教师看板
  scripts/           # simulate(全量仿真) / livetest(真实 API 全链路)
server/              # 零依赖 Node 网关:静态托管 + 登录会话 + LLM/语音转写代理 + 按账号存档
doc/                 # 方案文档(含密钥的文件均已 gitignore,永不入库)
```

## 安全纪律

- 一切真实密钥只存在服务器 `config.json`（0600）或本机 gitignored 文件，**前端产物与仓库零密钥**（每次发布均扫描验证）
- 登录墙：未登录仅可查看；聊天/转写/存档接口全部登录后按账号限流
- 学习存档按账号 sha256 落盘，密钥类设置永不进云端同步
