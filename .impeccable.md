# Design Context — 「小白同学」(学伴智仓/app)

## Target audience
- 主要:高校计算机/AI 课程学生(把知识讲给 AI 学生「小白」听的费曼反转学习者)
- 关键旁观者:竞赛评委(答辩演示 1440px 投屏,第一眼要读出"认真设计过",不是 AI 模板)
- 次要:教师(教师看板页,教务报表视角)

## Use cases
- 备课 → 课堂讲解(黑板对话) → 送考 → 复盘批注 → 成长册回看,一条师徒叙事主线
- 竞赛答辩现场演示与录屏;生产站点 http://106.53.163.57/xiaobai/

## Brand personality / tone
- 「书斋 × 少年气」:旧自习室的温暖 + 学生的认真,不是科技 dashboard、不是萌宠 app
- 视觉语言:月白瓷青纸底、靛青为骨、朱砂只做警示、印泥红只做印章(--seal-red 豁免)、黛绿=掌握、藤黄=衰减唯一暖色
- 版式:左对齐、内容列+眉批窄侧列(古籍批注构图)、每页一处破格瞬间
- 讲解舱是全站唯一深色场景(夜自习黑板 --board 系);页面侧深色锚点走 --inkstone 砚墨系,一页至多一两处
- 用户已明确否决:过度"AI 感"的暖奶油纸(2026-07-07 换色月白瓷青);2026-07-11 方向:保留书斋风,全站加浓加深(层次/对比/密度)

## Hard rules
- 完整设计法典见 app/DESIGN.md(违反=返工);令牌唯一来源 app/src/styles/tokens.css
- 改 tokens 中被 palette.ts 镜像的值必须同步重换算(WebGL sRGB 镜像)
- 入场动画 global(rise);往复动画 var(--ease-loop);断点 960/720/560;版心 var(--page-max)
