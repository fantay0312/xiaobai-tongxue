# 「小白同学」设计规范 —— 老学堂·邮驿票据

所有页面实现前必读。违反本文件的 PR 视为未完成。
(2026-07-22 全站改版:春雾书院退役,现行视觉=复古票据/老学堂文具质感,参考锚为 Post Post 旅宿票据设计稿)

## 定位与气质

高校学生把知识**讲给 AI 学生听**的严肃教育产品。气质:**老学堂 × 少年气**——像一间 1920 年代的老学堂文具铺:奶油纸、铅字打字机小签、邮戳圆章、听课证与借书卡、手贴的斜置票卡;不是科技感 dashboard,不是陪伴系萌宠 app。评委第一眼要觉得"这是被认真设计过的作品",不是"哪个 AI 生成的模板"。

## 硬性禁令(AI Slop 黑名单)

- 禁止紫蓝科技渐变、霓虹发光、暗黑模式全局主题、玻璃拟态卡片堆
- 禁止渐变文字、大圆角图标配标题的模板布局、千篇一律的卡片网格
- 禁止 emoji 当设计元素滥用(数据里已有的 🌟📋 除外)
- 禁止 bounce/elastic 缓动;只用 var(--ease-out)(往复动画见动效节豁免;邮戳匀速旋转 linear 属专用豁免)
- 禁止纯黑纯白;一切颜色从 tokens.css 取,不得新造十六进制色值
- 禁止 Modal 弹窗;用侧栏、抽屉、页面内展开
  - **弹层豁免(共四个)**:拜师帖(MentorLetter)、小白引路(Tour)、设置(SettingsDialog)、个人中心(ProfileDialog)——共同工艺:淡墨罩、Esc 可退、焦点圈、同屏只许一个(引路对其余弹层礼让)。此豁免不外延:新弹层默认仍然违禁

## 色彩纪律

- 底色永远是奶油纸 `--paper`(#F4F1EA 锚);分区在麦色纸 `--paper-warm`、陶粉纸 `--paper-blush`、暖灰纸影 `--paper-shade` 之间交替,不堆卡片阴影;界栏用暖灰 `--paper-edge`/`--line-strong`
- 主动作色为深赭陶 `--azure`(历史兼容名,= `--terra-ink` #9E4526);生赭陶 `--terra`(#C65D3B)对比度仅 3.7:1,**只许 ≥24px 大字、图形、色块底**,小字一律用 `--azure`
- **芥末黄 `--mustard`(#FFC233)是高光票卡色**:一页至多一张芥末票卡(hero 卡/关键证卡),墨字压其上(10.2:1);金印语境写 `--gold/--gold-ink`。衰减语义的赭金 `--amber` 比芥末沉一档,互不得混用
- 警示=绛红 `--cinnabar`,**只**给误区/盲区/警示——泛滥则失效;与赭陶动作色以深度+形制(chip/边线)区分
- **印章豁免**:红戳(品牌章、成就朱印、私章)写别名 `--seal-red`;墨线官戳(双圈虚线 OFFICIAL 制式、环形铅字盘)直接用墨系,两种戳都是旋转圆章形制,与警示 chip 不同形,语义不相扰
- 掌握=橄榄绿 `--jade`,衰减=赭金 `--amber`,未学=暖灰 `--dust`(盲区图谱三态)
- 深色锚点面板(账簿封皮/证书底/黑圆钮)走 `--inkstone` 暖炭系,一页至多一两处;**讲解舱是全站唯一深色页**:老教室青灰炭黑板,用 `--board/--chalk` 系,这是场景叙事,不是暗色模式

## 字体(四声部)

- 题头/大字/斜体点睛:`--font-serif`(Cormorant Garamond + 宋体);**衬线斜体是本站的"手写签名位"**(英文斜体、中文配楷体)
- 小白台词/手写批注/金句:`--font-display`(霞鹜文楷)——是"填进票据空格的手写字",不再做题头
- 打字机小签:`--font-typewriter`(Courier Prime + 宋体铅字),**一律大写+`letter-spacing: var(--track-label)`**——编号、眉批标签、索引、页脚小字全走这一档;公共类 `paper.module.css .typeLabel`
- 正文/控件:`--font-body`(DM Sans + 苹方);代码:`--font-code`
- 《学记》引文"教然后知困"是全站精神锚点,首页必须出现且排版考究(竖排宋体,公共件 anchor.module.css)

## 票据工艺(招牌元素,公共件 src/styles/paper.module.css)

- **纸纹**:票卡/色块盖 `.texture` 噪点浮层;全站底纹与胶片颗粒已在 index.css,页面勿重复铺
- **叠卡**:2-3 张票卡微斜叠放(`.tiltL/.tiltR`,±2-3°),hover 回正半度;**斜置层与 global(rise) 入场层必须分开**(to 帧 transform:none 会压死斜置)
- **邮戳**:环形铅字盘 `components/ui/RoundStamp`(慢转 20-60s)+ 双圈虚线官戳 `.stamp/.stampInner`;每页至多一枚转戳
- **撕票线**:`.perfLine` 虚线裁切位 + `.perfNote` 小字,用于票根/收据分节
- **竖排边栏字**:`.verticalRail` 贴票卡边缘的索书号/电话小字
- **检票口**:`.notch` 票卡顶缘半圆挂孔,只给"钥匙卡"档的主票卡
- 节制:一页至多一处叠卡阵 + 一枚转戳;工艺是佐料不是主菜

## 布局

- 左对齐为主,标题区允许大量留白;禁止一切居中堆叠(叠卡阵内部除外)
- 节奏:紧凑成组(--gap-tight 内),组间疏朗(--gap-loose / --gap-section)
- 不对称:内容列 + 窄侧注列(像票据的存根栏)是本站的招牌构图,复盘页/备课页优先采用
- 每页要有一个"非常规瞬间":一处打破网格的元素(斜置票卡、慢转邮戳、贴边竖排小字、超出栏宽的图)

## 动效

- 页面载入:一次编排好的 stagger 入场(60-90ms 间隔,translateY(8px)+opacity,--t-med --ease-out),之后克制
- 入场关键帧唯一定义在 index.css,模块内必须写 `animation: global(rise) ...`(裸名会被 CSS Modules 哈希导致动画失效);hover 元素配 backwards,勿用 both
- **缓动豁免**:往复环境动画(呼吸/浮动/眨眼)允许对称缓动,但必须写 `var(--ease-loop)`;邮戳匀速旋转写 `global(spinslow) … linear`;单向运动仍只用 `var(--ease-out)`
- 小白台词:打字机流式感(逐字或逐词,像"学生在想怎么说")
- 图谱点亮、出师仪式是两个允许铺张的高光时刻
- 只 transition transform/opacity;高度展开用 grid-template-rows

## 组件约定

- 样式一律 CSS Modules(`*.module.css`),类名语义化;公共令牌从 tokens.css 取;票据工艺从 paper.module.css 取
- 页面自包含:每页自己的小组件放自己目录,不建全局组件库(RoundStamp/Icon 属既有 ui 例外)
- 小白形象统一用 `components/xiaobai/XiaobaiAvatar`(props 契约冻结);2D 精灵图集,不得退回团子/科技机器人
- 状态只从 `store/appStore.ts` 取;页面不直接 import engine
- 页根必须写 `width: 100%`(AppShell .main 纵向 flex 下 margin:0 auto 子项会 shrink-to-fit 压瘦整页)

## 各页艺术方向一句话

- **宣传页**:旅宿前台。hero=三张斜置叠放的票卡(芥末钥匙卡+赭陶邀请卡+暖灰收据卡),下方三栏小图标服务卡
- **首页**:学堂门厅。《学记》引文 + 听课证台面;书架是贴了书脊题签的老书柜
- **备课页**:自习桌面。讲义像铅字油印稿,材料包像牛皮纸档案袋,索引走打字机小签
- **讲解舱**:老教室青灰黑板。左 1/3 黑板前的书童小白,右 2/3 对话;导演动作以粉笔小字注在消息旁
- **复盘页**:一叠批改后的收据。红笔批注感,高光在前,证据链像票根可展开,出师=盖官戳
- **成长页**:学籍档案册。图谱是星图,印匣是一页邮票册,证书带钢印质感
- **教师看板**:教务登记簿。克制、数据密度高,"讲不清 Top5"用墨条横图
- **登录页**:拜师帖=一张待检的听课证(检票口挂孔+撕票线)
